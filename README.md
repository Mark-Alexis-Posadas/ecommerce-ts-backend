# 🚀 MERN E-Commerce API (TypeScript)

A full-featured **E-Commerce Backend API** built with **MongoDB, Express, Node.js, and TypeScript**.

---

## 📦 Features

- 🔐 JWT Authentication
- 👤 User Management
- 🛍️ Product API (with pagination)
- 🛒 Cart System (Add, Increment, Decrement, Remove)
- 📦 Order Management
- 🌐 CORS Enabled
- ⚡ RESTful API

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- TypeScript
- MongoDB + Mongoose
- JWT

---

## ⚙️ Installation

```bash
git clone <your-repo>
cd backend
npm install
```
